Spacing

The space padding class offers a consistent group of options to apply white space above, below, or above and below content separate from the grid system or custom CSS.

Using this group of consistent sizes (based on the grid system $grid-gutter value) maintains consistency throughout a design while offering considerable flexibility.

While this class is does not offer responsive breakpoint options, it is fluid from small screens up to the website max width, at which point it will lock to the defined value.

Usage

Classes can be combined to create various configuations:

Apply normal padding with small bottom padding
class="pad pad-s-b"
Apply small bottom padding with large top padding
class="pad-s pad-l-t"
Just bottom padding
class="pad-b"
