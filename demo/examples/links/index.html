Links & Navigation

For years the traditional mathod of styling links would inevitably over time drive me insane with specificity problems and overrides. So eventually I came to the conclusion that links should simply be scoped, via classes.

This does mean that you have to be sure to catch every link in your templates, otherwise they'll appear unstyled. It also means you'll be using unsemantic blah blah blah classes. However it does produce easier to maintain code with fewer long-term headaches. Just create a link style on a class and stick it where it needs to go in your markup (be sensible!).
